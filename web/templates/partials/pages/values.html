Values and Types

Primitive Types

Ego has a small set of built-in value types. More complex data structures are built on top of these primitives.

Integer

Whole numbers without a fractional component. Examples: 0, 42, -7.

Float

Numbers with a decimal point. Examples: 3.14, 0.0, -1.5.

String

Text values enclosed in single quotes. Examples: 'hello', ''.

Boolean

Logical truth values: true and false.

Nil

A special value that represents "no value" or "nothing". It is distinct from false and from an empty string.

Dynamic Typing

Ego is dynamically and strongly typed. Values carry their type at runtime, and variables can hold any type, but invalid operations are still rejected.

dynamic.ego
example
x := 41      # integer
x = 3.14     # now a float
x = "hello"  # now a string

# still strong typing: this is an error
x = "hi" + 1  # cannot add string and integer
Runtime types. Each value knows its type at runtime. Operations check these types when you execute the program.
Type changes. The same variable can hold an integer, then a string, then a boolean. Rebinding a different type is allowed.
Strong typing. Ego does not silently coerce unrelated types. You cannot, for example, add a string and an integer without converting one side explicitly.
Invalid operations. When you try to use an operator with incompatible types, Ego reports a runtime error instead of guessing what you meant.

Truthiness

In conditional contexts, some values count as "true" and others as "false". This is called truthiness.

truthiness.ego
example
if 0 
  put('0 is truthy')
;

if ''
  put('empty string is truthy')
;

if nil 
  put('this never runs')
;
True values. All values are truthy except a small set of falsey values. Non-zero numbers, non-empty strings, and any other non-nil value evaluate to true.
False values. The boolean false and nil are falsey. They cause conditions to be skipped.
Nil behavior. nil represents "no value" and is always falsey in conditionals. It is not equal to false, but behaves similarly when used in if and other checks.
