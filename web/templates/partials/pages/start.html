Install

Ego is still in early development, so installation typically means building from source (for now). You will need to have the Go compiler installed.

Build from source
example
# clone the repo
git clone https://github.com/taus9/ego.git
cd ego

# build
go build

# run
./ego version
Tip: put the built binary somewhere on your PATH so you can run ego anywhere.
